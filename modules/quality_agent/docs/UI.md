# 前端审核界面展示项（单篇笔记）

本节定义单条笔记在前端审核界面中需展示的主要内容，便于审核人员快速判断文本质量与合规风险，并进行优化或标注。该设计对应于“文本质量 Agent”与“合规风险 Agent”的输出结构，建议采用模块化卡片 + 高亮提示 + 可展开细节的方式呈现。

## 1. 文本质量类展示项

| 项目 | 展示字段/建议形式 | 说明 |
| --- | --- | --- |
| 查重率 | 本批次查重率 / 历史查重率 / 是否命中标红 | 命中阈值需高亮提示，可展开查看重复原文 pair。 |
| 内容潜力总分 | 总分 / 评级 / 状态标签 | 支持展开查看评分细项。 |
| 潜力分析细项 | 结构规范分 / 相似度分 / 热点覆盖分 | 展示权重与解释信息。 |

## 2. 合规风险类展示项

| 类别 | 展示字段/建议形式 | 说明 |
| --- | --- | --- |
| 合规风险总览 | 风险总分 / 等级标识 / 风险类型分布 | 支持点击查看详情。 |
| 黑名单命中项 | 风险词 / 位置 / 风险等级 / 违规类型 | 高风险词高亮，支持定位原文。 |
| 句段级风险提示 | 风险句段编号 / 识别类型 / 建议处理方式 | 多义/语义风险需人工复核提示。 |
| 用词规范建议（可选） | 推荐项 / 替换词 / 高亮位置 | 辅助优化表达。 |
